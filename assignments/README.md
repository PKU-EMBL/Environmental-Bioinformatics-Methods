# 方法训练总览 | Practice Library

本目录提供四个非考核训练模板，用于自学、课堂讨论或准备课程汇报。当前课程考核只有课程汇报；除非任课教师另行说明，这些任务不单独计分，也不要求统一提交。

## 任务索引

| 任务 | 训练重点 | 建议产出 | 对应课程 |
| --- | --- | --- | --- |
| [P0 · 计算环境与可复现性](hw0_environment.md) | 环境锁定、命令记录、最小测试 | 环境文件 + 运行记录 | 全课程 |
| [P1 · 序列相似性与注释](hw1_similarity.md) | 搜索设计、命中解释、证据边界 | 命中表 + 解释备忘录 | L2–L3 |
| [P2 · 宏基因组流程设计](hw2_metagenome_pipeline.md) | 组装、分箱、质控与失败诊断 | 流程图 + 决策表 | L4–L8 |
| [P3 · 生态解释与报告](hw3_ecology_reporting.md) | 指标选择、可视化、克制解释 | 一页分析简报 | L9–L10 |

## 建议用法

1. 从课程汇报中最不确定的方法节点选择一个任务。
2. 使用小数据或公开示例完成最小可验证版本。
3. 记录工具之外的判断：为什么这样选、哪里可能失败、结果不能说明什么。
4. 把最有解释力的一张图、一个表或一个流程节点汇入最终汇报。

训练的完成标准不是“软件运行成功”，而是留下可核查的输入、过程、输出与判断。
