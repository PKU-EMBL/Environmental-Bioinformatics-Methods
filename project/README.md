# 课程汇报指南 | Course Presentation Guide

## Scope

围绕一个环境生物信息学问题、方法链条或课件相关案例，完成一场可答辩的课程汇报。当前课程考核只有课程汇报，不再设置单独项目设计书或阶段性项目分数。

## Suggested Working Folder

```text
presentation-team-name/
├── README.md
├── data-notes/
├── env/
├── workflow/
├── figures/
└── slides/
```

## Minimum Quality Bar

- 问题定义清楚，范围适合课堂汇报。
- 数据、数据库、软件版本和关键参数可核查。
- 方法选择有理由，能够连接课程中的核心概念。
- 图表服务于论点，而不是堆叠输出。
- 汇报包含局限性、失败风险或替代解释。
